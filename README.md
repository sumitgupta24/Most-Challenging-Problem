## Overview

Hush is a real-time chat application built with modern web technologies, offering fast, reliable, and seamless messaging between users.

---

## Most Challenging Problems I Solved Recently

* Implementing smooth and reliable real-time chat using Socket.io in the initial stages.
* Identifying and fixing dropped messages during high-speed messaging between users.
* Resolving issues caused by unoptimized socket event handling and missing acknowledgements.
* Restructuring the message flow to prevent race conditions and ensure message delivery.
* Introducing a lightweight retry mechanism to guarantee consistent, stable communication.
